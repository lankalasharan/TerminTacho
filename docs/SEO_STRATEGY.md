## 🎨 On-Page SEO Checklist

### Every Page Must Have:

#### 1. **Title Tag** (50-60 characters)
```
Format: [Primary Keyword] | TerminTacho
Example: Blue Card Germany Processing Times | TerminTacho
```

#### 2. **Meta Description** (150-160 characters)
```
Include: Keyword + Call to action + Unique value
Example: "See real Blue Card processing times from 1000+ users across Germany. Submit your timeline and help others. Free crowdsourced data."
```

#### 3. **H1 Tag** (Only one per page)
- Include primary keyword
- Make it compelling
- 20-70 characters

#### 4. **URL Structure**
```
Good: /blog/blue-card-processing-time-germany
Bad: /blog/post-123
```

#### 5. **Image Optimization**
- Alt text with keywords
- WebP format
- Compressed (<100KB)
- Descriptive filenames

#### 6. **Internal Links**
- Link to related blog posts
- Link to city pages
- Link to submit form
- Use descriptive anchor text

#### 7. **Schema Markup**
- Article schema for blog posts
- Organization schema on homepage
- FAQ schema on FAQ page
- Breadcrumb schema everywhere
